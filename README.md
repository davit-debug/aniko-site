# Aniko — Anna Chikhiashvili personal brand site

Playful deep-indigo personal site: hero (Canva-based illustration), About Me,
3 collapsible services, brands wall, contact footer + Monthly Content Calendar
(Instagram / LinkedIn plan).

- `content-calendar.html` — content calendar (edit the `CAL` data array in the inline script to add months/posts)

Built with plain HTML/CSS/JS. Fonts: Syne · Fredoka · Montserrat · Noto Sans Georgian.
